Text-to-SQL with Guardrails

Featured · LLM Safety

Natural-language → SQL where the LLM writes the query but a sqlglot AST guardrail validates it (read-only, single-statement, table-allowlist, no file functions) before any row is read, then runs on DuckDB. Self-correction retry + execution-accuracy eval.

Semantic Cache for LLMs

Featured · LLM Infra

Drop-in cache that keys on prompt embeddings, not exact text — paraphrased repeat questions hit the cache (cosine ≥ 0.85) and skip the LLM call to cut latency and token cost. O(1) hash + vectorized cosine, LRU + TTL, 13 offline tests.

PythonEmbeddingsSemantic CacheOpenRouterNumPypytest
Live Demo GitHub
